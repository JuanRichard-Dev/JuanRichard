# V10.4.1 Interactive Hover

## Melhorias funcionais
- Hover reativado nos gráficos para inspeção dos dados.
- Zoom, arraste, seleção, edição e barra de ferramentas continuam bloqueados.
- Tooltip unificado em séries temporais.
- Histórico mostra valor observado, mês anterior e variação mensal.
- Projeção mostra estimativa e faixa inferior/superior quando disponível.
- Linha-guia vertical discreta para facilitar a leitura do período.
- Cursor de inspeção nos gráficos, sem adicionar novos elementos visuais permanentes.
